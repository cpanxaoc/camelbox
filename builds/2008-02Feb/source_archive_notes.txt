Instructions on where to put which files from each project's distribution
archives.

UnxUtils
========
- create unxutils\usr\share\unxutils\
- move StdDisclaimer.html\UnxUtilsDist.html to unxutils\usr\share\unxutils\
- move the contents of unxutils\usr\local\share\ to unxutils\usr\share\unxutils\
- move unxutils\usr\local\md5sum to unxutils\usr\share\unxutils\
- move unxutils\usr\local\wbin to unxutils\usr\local\bin

7zip (Optional)
===============
- create 7zip\bin, 7-zip\usr, 7-zip\usr\share\7zip
- move 7za.exe to 7zip\bin
- move 7zip.chm, license.txt, readme.txt, copying.txt to 7-zip\usr\share\7zip

dmake
=====
- create dmake\bin, dmake\usr, dmake\usr\share\dmake
- move dmake.exe and support\ to dmake\bin
- move all of the other files to dmake\usr\share\dmake

MinGW
=====
- Make sure you select g++
- Install to C:\camelbox
- move camelbox\COPYING* to camelbox\usr\share\MinGW
- move camelbox\MinGW shortcut to camelbox\usr\share\MinGW
- move installed.ini, MinGW-5.1.3.exe and uninst.exe to camelbox\mingw

zlib
====
- move the zlib .txt files to share\zlib-docs in gtk-support-dev

gtk+
====
- move share\gtk-doc from gtk+ zipfile to a new gtk-doc\ folder\archive
